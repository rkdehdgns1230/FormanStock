# For(Four)manStock

2022년도 2학기 데이터베이스 및 시각화 팀 프로젝트 레포지토리 

## 사용한 기술 스택 정리

### Front-end
1. HTML, CSS, JavaScript
2. BootStrap

### Back-end
1. Node.js
2. Express web framework
3. MySQL DBMS
4. pug template engine

### 세부 모듈 종속성 정보
